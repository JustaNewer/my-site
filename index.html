AI matching / Full stack

RoommateDistributor

GitHub

A full-stack dormitory operating system that combines deterministic personality modeling, hard schedule constraints and LLM reasoning to form more compatible rooms.

9D to OCEAN

Nine Likert dimensions are converted into a deterministic Big Five profile, so identical inputs always receive a consistent interpretation.

Three-layer allocation

Schedule clustering narrows the solution space, AI optimizes personality compatibility, then each room receives an explainable report.

Safe fallback

If the AI service fails, the system preserves the deterministic schedule-based allocation instead of blocking the workflow.

Operations interface

Administrators can inspect floors, rooms and beds, review compatibility, and drag, swap or remove occupants manually.
