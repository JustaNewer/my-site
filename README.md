# Jack Ye - Personal Site

A single-file, zero-dependency personal website with a strict black-and-white interface and original-color photography.

## Sections

- Home: avatar, public profile information, personal tags, and motto
- Resume: language, internship, project, and skill highlights
- Articles: reserved for future writing
- Solo Trip: Singapore, Osaka, Kyoto, Mount Fuji, and Tokyo photo journals

## Features

- Chinese and English interface
- Light and dark themes
- Hash-based section navigation
- Responsive desktop and mobile layouts
- Early-web-inspired square borders, monospace typography, and minimal motion

Open `index.html` directly, or serve the folder with any static web server.

For step-by-step publishing instructions, see [DEPLOYMENT.md](DEPLOYMENT.md).
